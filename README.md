# Google Drive Sync Client (Yandex.Disk Style GUI)

Аналог Яндекс.Диска для работы с Google Drive на базе PyQt6. Приложение сочетает в себе удобство облачного хранилища Google Drive и привычный, премиальный интерфейс Яндекс.Диска.

## Основные возможности

* **Стильный интерфейс**: Светлая тема, полностью повторяющая дизайн-систему Яндекс.Диска (шрифты, цвета, скругления, иконки, кнопка покупки места).
* **Синхронизация в реальном времени**: Автоматическая двусторонняя синхронизация локальной папки и Google Drive в фоновом потоке.
* **Интеграция с системным треем**: Удобное контекстное меню, кастомные динамические иконки статуса синхронизации (синхронизировано, синхронизируется, пауза, ошибка).
* **Навигация по папкам**: Локальный просмотр структуры каталогов до синхронизации и управление удаленными файлами.
* **Глобальные горячие клавиши**: Захват экрана и интеграция с редактором скриншотов.
* **Удобное управление аккаунтом**: Быстрый вход и выход (смена аккаунта) прямо из интерфейса.

## Требования

* Python 3.10+
* PyQt6
* Google API Client Libraries (`google-api-python-client`, `google-auth-oauthlib`, `google-auth-httplib2`)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd <папка-репозитория>
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python src/main.pyw
   ```
